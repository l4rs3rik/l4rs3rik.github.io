# ABL skins

## Instruction 
Download, unzip and copy the folder and the .abl3skin file found in the ZIP-file to your ABL3.2 
installation folder.

Windows:
c:\Users\<your user name>\Documents\AudioRealism\ABL3\

MacOs:
/Library/Application Support/AudioRealism/ABL3/
